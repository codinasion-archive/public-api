# Public API

## Free and Open Source fake API for testing and prototyping.

---

**Public API is a free and Open Source online REST API that you can use whenever you need some fake data.**

**It can be used to _prototype your application_,
_test your application_ without the need of a real server,
for a _demo on [CodeSandbox](https://codesandbox.io/)_, in _code examples on [Stack Overflow](https://stackoverflow.com/)_,
or simply to _test things locally_.**

---

```js fetch
fetch("https://pubapi.vercel.app/api/person")
  .then((response) => response.json())
  .catch((error) => console.log(error));
```
